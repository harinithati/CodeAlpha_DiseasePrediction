# CodeAlpha_DiseasePrediction
# ❤️ Heart Disease Prediction System

## 📌 Project Overview
This project predicts the likelihood of heart disease using Machine Learning algorithms and clinical patient data.

The application is built using:
- Python
- Scikit-learn
- XGBoost
- Streamlit

---

## 🚀 Features

✅ Real-time Heart Disease Prediction  
✅ Interactive Professional Web App  
✅ Machine Learning Based Analysis  
✅ High Accuracy Prediction Model  
✅ User-Friendly Interface  
✅ Live Deployment using Streamlit Cloud  

---

## 🧠 Machine Learning Models Used

- Logistic Regression
- Random Forest
- XGBoost Classifier

### ⭐ Best Model:
XGBoost achieved the highest accuracy.

---

## 📊 Dataset

Heart Disease UCI Dataset

Dataset includes:
- Age
- Cholesterol
- Blood Pressure
- Chest Pain Type
- ECG Results
- Heart Rate
- And other medical parameters
